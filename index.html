Contributing

If you are interested in the current development status, you can just take a look at our issue tracker. If you find a bug or have an idea how we could make PHProjekt better, you can tell us there.

Mailing list

We use a mailing list to discuss development. You can browse the archive or subscribe at the info page if you are interested in the development of PHProjekt.

Version control and github

We are using git as our version control system. You can directly clone git://github.com/Mayflower/PHProjekt.git or simply have a look at our github page. We would love to see pull requests.

Modules

If you have created a module that you want to share with other PHProjekt users, please contact us and we will put it up here.

Community

PHProjekt is a community project. This shows through an open infrastructure and community platforms. You can visit the forum for discussions and help or just browser current issues and comment there.

Key features

PHProjekt is a web application that helps you with planning and running your projects.

Download archive

All released versions of PHProjekt can be downloaded here. If you are looking for older versions head over to http://p5.phprojekt.com. Please note that for various reasons, you should not use PHProjekt 5 anymore.

Demo

If you are interested in PHProjekt but not yet convinced, take a look at our live demo. It is running the latest stable version.

News

We are currently in the middle of a very productive week working of full-time work on PHProjekt. This includes the new website design and the release of the first alpha version of PHProjekt 6.1.0. The release will include a new calendar, the possibility to include subprojects in listings of todos, notes, etc. and a huge reduction in client memory usage.
We have been planning other great features for the near future, so don't forget to take a look from time to time.
